# Yandex-Gallery
Приложение для школы мобильной разработки Яндекса 2018
## Features
* Обработка изменений конфигурации.
* Пагинация данных.
* Кэширование данных.
* Unit тесты.
## Использованные библиотеки
* Retrofit 2
* Butter Knife
* RxJava 2
* Dagger 2
* Glide
* PhotoView
* Paper
